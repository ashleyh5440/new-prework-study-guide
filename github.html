GitHub

GitHub is a developer platform that allows developers to create, store, and manage their code.

To create a new repository:

  • click New button
  • input a name and description
  • check the Add a README file
  • choose Add .gitignore option if needed for your project
  • choose a license from the options
  • click Create repository

To clone a repository:

  • click the repository name
  • click the Code button
  • under Local tab, choose SSH and copy the path
  • in GitBash or terminal, type git clone and paste the GitHub path after
