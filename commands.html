Commands

  • pwd current working directory
  • ls list of files/folders
  • cd takes you to home directory
  • cd [destination] move between directories
  • cd .. navigate backwards in directory
  • cd [destination] move between directories
  • cd .. navigate backwards in directory
  • mkdir [directory name] creates new directory
  • rm -rf [directory name] deletes directory
  • touch [file name] creates file in directory
  • rm [file name] deletes file in directory
  • start opens a file
  • git status shows current branch, checks if you're in a repo
  • git checkout -b [branch name] creates new branch
  • git clone [paste SSH key or HTTPS]
  • clones repo to machine
  • git pull origin [branch name] syncs local branch with main branch
  • git push origin [branch name] pushes local changes to GitHub branch
  • git commit -m "[comments]" adds commits to the repo
  • To push local changes to GitHub, do the following steps:
    1. git add -A
    2. git commit -m "[comments]"
    3. git push origin [branch name]
